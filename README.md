# Database-Normalization-Visualizer
Teaching Tools : Database-Normalization

- Step-by-step visualization for UNF, 1NF, 2NF, and 3NF.
- Bilingual support (English and Traditional Chinese) 
- Educational context explaining the "why" behind normalization (redundancy, anomalies, partial/transitive dependencies).
